# 日比整体 だいせんじ接骨院 リニューアルサイト

## 概要
本リポジトリは、**日比整体 だいせんじ接骨院**の公式サイトをリニューアル制作したWebサイトのソースコードです。
既存サイトの課題を整理し、情報の分かりやすさと信頼感の向上を目的として制作しました。
進級制作作品として取り組んだものです。

🔗 **サイトURL**
https://land-jj22.vivian.jp/hibiseitai/

---

## 制作背景・課題
既存サイトには以下のような課題がありました。

- 整体と接骨院の違いが分かりづらい
- 情報が整理されておらず、目的のページにたどり着きにくい
- 更新作業のハードルが高い

クライアントからは、
「信頼感のあるデザイン」「強みが伝わるサイト」「更新しやすい仕組み」
を求められていました。

---

## 解決策・工夫したポイント

### 信頼感を意識したデザイン
- ブランドカラーの**黄色**を基調に、安心感のある配色を設計
- 文字色に黒ではなく**紺色**を使用し、柔らかく落ち着いた印象に
- サイト内で使用する写真はすべて撮影し、世界観を統一

### 整体と接骨院の違いを明確化
- トップページで違いを分かりやすく解説
- 「保険適用」「交通事故治療」など、利用者が迷いやすいポイントを整理
- 個別ページで施術内容やスタッフ紹介を詳しく掲載

### 更新しやすいサイト設計
- WordPressを導入
- カスタム投稿タイプ・カスタムブロックを作成し、
  専門知識がなくても料金表やお知らせを更新可能に

### 情報設計・導線設計
- ファーストビューに主要ページへの導線を集約
- スクロール中も迷わないナビゲーション設計
